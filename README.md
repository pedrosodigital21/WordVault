## 📖 What is WordVault?

WordVault is a simple and clean web app that helps you store words and their translations. You can search your saved words, hide or show translated words, and export your list to a JSON file. It works without ads or distractions so you can focus on building your vocabulary.

This tool is designed for anyone learning a language or wanting to keep track of new words. You don’t need any technical skills to use it.

## 📂 Exporting Your Words List

You can export your saved words to a file for backup or sharing:

1. Click the "Export JSON" button.
2. Your browser will create a `.json` file that contains your vocabulary list.
3. Save this file to your computer.
4. You can reopen this file later or use it in other apps that support JSON.

This process allows you to keep your word list safe and upload it again if needed.
